# GNO
###########
SCRIPTS
###########
NO.py: general script that combines different types of neural operators
xFNO.py: Fourier Neural Operator (FNO) and Physical Fourier Neural Operator (PFNO). The latter imposes conservation laws
xWNO.py: same as above but using SWT instead of FFT
layers.py: contains layers used to build the neural networks (fourier, wavelet, activation, mass conservations)
special_loss_functions.py: contains some custom loss functions better suited for learning PDEs


###########
DIRECTORIES
###########
solver: PDE solvers
data: timeseries generated using solvers
plots: some gifs generated by the jupyter notebooks
